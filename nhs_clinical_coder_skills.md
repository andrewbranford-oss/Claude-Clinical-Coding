# Skills Required for NHS Clinical Coding Check

This document outlines the essential skills and competencies required to perform effective clinical coding checks within the NHS context.

---

## Core Skills

- **ICD-10 Coding Proficiency**
  - Accurate assignment of diagnostic codes using ICD-10 (5th Edition)
  - Understanding of dagger/asterisk conventions, sequencing, and coding rules
  - Ability to verify candidate codes in the NHS England Classifications Browser at https://classbrowser.nhs.uk/

- **OPCS-4 Coding Proficiency**
  - Accurate assignment of procedure codes using the OPCS-4 version mandated for the episode date
  - Application of laterality, imaging, and anaesthesia codes
  - Ability to verify OPCS-4 code descriptions, notes, inclusions, exclusions, and version/year in the NHS England Classifications Browser

- **Clinical Knowledge**
  - Ability to interpret clinical documentation (discharge summaries, operation notes, clinic letters)
  - Understanding of common medical terminology, abbreviations, and clinical pathways

- **Spreadsheet Data Handling**
  - Ability to ingest approved patient-data `.xlsm` workbooks without executing embedded macros
  - Maintenance of workbook, worksheet, row, and column provenance for every extracted patient or episode field

- **NHS Coding Standards**
  - Familiarity with NHS Clinical Coding Standards (NCCS) and specialty-specific guidance
  - Adherence to Information Governance, GDPR, and Data Security Toolkit requirements

- **Data Validation & Quality Assurance**
  - Checking code assignments against NHS Data Dictionary and local rules
  - Use of HRG Grouper for validation and tariff assignment
  - Identifying and resolving discrepancies or incomplete documentation

- **Audit & Traceability**
  - Maintaining comprehensive audit trails for all coding actions
  - Logging amendments, escalations, and coder reviews

- **Workflow & Exception Handling**
  - Managing workflow queues, case routing, and escalation procedures
  - Handling exceptions such as ambiguous, conflicting, or missing data

- **Communication & Collaboration**
  - Raising clinical queries with responsible clinicians
  - Collaborating with coding teams, auditors, and managers

---

## Desirable Skills

- Experience with NHS PAS/EPR and coding software
- Familiarity with digital dictation/transcription systems
- Knowledge of local trust policies and audit requirements
- Continuous professional development in clinical coding
